# QGraphicsItem自定义图元示例

## 简介
本仓库提供了一个自定义QGraphicsItem图元的示例源码demo。该示例展示了如何创建自定义图元，并实现拖动绘制、拖动拉伸缩放图元的功能。

## 功能特点
- **自定义图元**：创建自定义的QGraphicsItem图元。
- **拖动绘制**：支持通过拖动鼠标在场景中绘制图元。
- **拖动拉伸缩放**：支持通过拖动图元的边界进行拉伸和缩放操作。
